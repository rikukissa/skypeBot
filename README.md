# Python SkypeBot

## Requirements

* Running Skype client
* Python 2.x

## Usage
Rename the settings.example.py file to settings.py

    $ pip install Skype4Py
    $ python skypeBot.py

## Commands
You can send the messages directly to the bot client or use the commands in a group conversation
